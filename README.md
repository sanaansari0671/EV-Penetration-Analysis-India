# EV Penetration Analysis in India

## Project Overview

This project analyzes Electric Vehicle (EV) adoption trends across Indian states using data analytics and business intelligence techniques. The objective is to identify growth patterns, vehicle category trends, and state-wise EV penetration using interactive dashboards and visual reports.

## Objectives

* Analyze EV sales and registrations across Indian states.
* Identify top-performing states in EV adoption.
* Study trends across vehicle categories.
* Develop interactive dashboards for business decision-making.
* Present actionable insights using Power BI and data analytics.

## Tools & Technologies

* Python
* Jupyter Notebook
* Power BI
* Streamlit
* Pandas
* NumPy
* Data Visualization

## Project Files

* EV_Sales_Master.csv
* Ev_monthlywith_category.csv

## Key Insights

* EV adoption has shown significant growth in recent years.
* Electric Two-Wheelers dominate the market share.
* State-wise adoption patterns vary considerably across India.
* Data-driven dashboards help identify high-growth regions and market opportunities.
